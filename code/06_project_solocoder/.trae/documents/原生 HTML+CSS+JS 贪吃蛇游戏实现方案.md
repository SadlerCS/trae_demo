## 功能概览
- 计分：吃到食物加分，局内显示当前分数；可选记录最高分（`localStorage`）。
- 背景：提供多种内置主题，通过下拉框即时切换（纯色、棋盘、网格、渐变、星空）。
- 运行：打开浏览器即可玩，键盘控制与移动端触控按钮。
- 代码：单文件，包含 `<style>` 与 `<script>`，核心逻辑配中文注释。

## 技术方案
- 使用 `<canvas>` 绘制栅格地图与蛇体、食物；避免外部依赖。
- 栅格单位 `cellSize=20`，画布约 `600x400`（30×20 格），保证清晰像素对齐。
- 主循环基于固定时间步（如 `120ms`），用 `setInterval` 控制节奏，保证一致性。
- 数据结构：蛇用坐标数组，方向队列防止瞬间反向；食物随机生成且避开蛇身。

## 页面结构
- 顶栏：标题、当前分数、最高分、背景主题选择、难度选择（速度档位）、开始/暂停、重新开始。
- 画布区：居中显示游戏区域，主题作为容器背景，Canvas 用透明/浅色网格（可开关）。
- 提示区：键位说明与移动端触控按钮（四向）。

## 游戏机制
- 移动：按当前方向在栅格中推进；吃到食物不移除尾巴段，实现“成长”。
- 碰撞：撞墙或撞到自身判定失败，停止循环并显示提示；支持一键重开。
- 食物：在空格随机生成，避免与蛇身重叠；可添加简单视觉区分（颜色或形状）。
- 难度：通过选择速度（如慢/中/快）影响步进间隔。

## 背景主题
- 纯色：柔和 UI 背景。
- 棋盘：两色交替格子。
- 网格：细线网格，便于观察移动。
- 渐变：霓虹或清新渐变（CSS 线性/径向渐变）。
- 星空：CSS 背景 + 随机星点（仅装饰，不影响 Canvas 绘制）。

## 控制与交互
- 键盘：方向键或 `W/A/S/D`；避免 180° 反向导致自撞（同帧忽略反向输入）。
- 触控：四向按钮，点击后立即变更方向；按钮大小适配手机。
- 开始/暂停：切换主循环运行状态；重新开始重置状态与分数。
- 提示：`Game Over` 弹层或文本，显示最后得分与最高分。
